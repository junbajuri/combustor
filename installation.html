InstallationÂ¶

  1. Download CodeIgniter here and extract it to your web server.
  2. Configure the database connectivity settings in application/config/database.php.
  3. Install Combustor through Composer:
$ composer require rougin/combustor --dev
  1. Choose if you want to install Wildfire or Doctrine ORM or both:
$ php vendor/bin/combustor install:wildfire
$ php vendor/bin/combustor install:doctrine
