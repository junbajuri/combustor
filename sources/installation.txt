Installation
============

1. Download CodeIgniter `here`_ and extract it to your web server.

2. Install ``Combustor`` using `Composer`_::

	$ composer require rougin/combustor

3. Choose if you want to install Wildfire or Doctrine ORM or both::

	$ php vendor/bin/combustor install:wildfire
	$ php vendor/bin/combustor install:doctrine

4. Lastly, configure the database connectivity settings in ``application/config/database.php``.

.. _Composer: https://getcomposer.org/
.. _here: https://github.com/bcit-ci/CodeIgniter/archive/3.0.0.zip
